# Pad Manager Checklist

*Role: Main manager of the static fire pad. Oversees all operations and has main contact with mission control manager.*

## Pad Setup
- [ ] Confirm tested system is sufficiently far or out of LOS.
- [ ] **TELL OPERATIONS MANAGER (SAFETY) TO COMMENCE CHECKOFFS.**
- [ ] Clear extraneous materials and personnel from static fire pad.
- [ ] Establish and maintain main contact with Mission Control Manager.
- [ ] Secure items and fittings (gauges, hose lines, harnessing).
- [ ] Confirm hardware is fixed (propellant tanks, relief valve outlets).
- [ ] Confirm GSE is set up safely with GSE Manager.
- [ ] **TELL MISSION CONTROL MANAGER PAD IS SECURE.**

## Fuel Fill Phase
- [ ] **BREAK HERE. WAIT FOR OPERATIONS MANAGER TO GIVE THE GO TO BEGIN FUEL FILL.**
- [ ] **TELL ETHANOL FILL OPERATOR TO COMMENCE FUEL FILL.**
- [ ] Monitor Fuel Fill operations on pad.
- [ ] Confirm Fuel Transfer Tank Vent is closed.
- [ ] Return system to Armed state after fill via Mission Control.

## Cryogen Fill Phase
- [ ] **TELL LOX FILL OPERATOR TO DON CRYOGEN PPE.**
- [ ] **TELL AVIONICS MANAGER TO ENERGIZE DAQ SENSE POWER.**
- [ ] **TELL ECOMM/ACTCOMM TO CONDUCT COMMS CHECK.**
- [ ] **TELL LOX FILL OPERATOR TO BEGIN LOX DEWAR CONNECTION.**
- [ ] Monitor Cryogen Fill on pad.
- [ ] **BREAK HERE. WAIT FOR LOX FILL OPERATOR TO CONFIRM FILL.**

## Pad Closeout
- [ ] **TELL AVIONICS MANAGER TO ENERGIZE DAQ POWER AND MONITOR CURRENT.**
- [ ] **TELL AVIONICS MANAGER TO PERFORM REMOTE SIGNAL CHECK.**
- [ ] **TELL AVIONICS MANAGER TO DE-ENERGIZE GROUND SYSTEM.**
- [ ] **TELL GN2/HIGH PRESS FILL OPERATOR TO SET REGULATORS.**
- [ ] **TELL AVIONICS MANAGER TO ENERGIZE SYSTEM.**
- [ ] **TELL MISSION CONTROL MANAGER ALL PAD TEAMS ARE CLEAR.**
- [ ] **BREAK HERE. WAIT FOR HOTFIRE COMPLETION.**

## System Safing
- [ ] **BREAK HERE. WAIT FOR MISSION CONTROL MANAGER TO CONFIRM SYSTEM BELOW 25 PSI.**
- [ ] **TELL AVIONICS MANAGER TO DE-ENERGIZE POWER ELECTRONICS.**
- [ ] Inspect fluid system components on static fire pad.
- [ ] **TELL MISSION CONTROL MANAGER PAD IS INSPECTED AND SAFE.**
