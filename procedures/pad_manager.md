# Pad Manager Checklist

*Role: Main manager of the static fire pad. Oversees all operations and has main contact with Mission Control Manager.*

## Pad Setup
- [ ] Confirm tested system is sufficiently far or out of LOS.
- [ ] Confirm all pad personnel are wearing required PPE per role.
- [ ] **TELL OPERATIONS MANAGER (SAFETY) PAD IS READY FOR SAFETY OFFICER CHECKOFFS.**
- [ ] Clear extraneous materials and personnel from static fire pad.
- [ ] Secure non-essential gauges, hose lines, and avionics harnessing.
- [ ] Confirm propellant tanks are strapped down.
- [ ] Confirm relief valve outlets are directed away from personnel and property.
- [ ] Confirm mounting hardware is torqued and secure.
- [ ] Confirm pressurized bottles are strapped down.
- [ ] Confirm GSE is set up safely with GSE Manager.
- [ ] Establish and maintain main contact with Mission Control Manager.
- [ ] **TELL MISSION CONTROL MANAGER PAD IS SECURE.**

## Fuel Fill Phase
- [ ] **BREAK HERE. WAIT FOR OPERATIONS MANAGER TO GIVE THE GO TO BEGIN FUEL FILL.**
- [ ] **TELL ETHANOL FILL OPERATOR TO COMMENCE FUEL FILL PER MASTER PROCEDURE.**
- [ ] Monitor Fuel Fill operations on pad.
- [ ] Confirm proper valve alignment before pressurization.
- [ ] Monitor for leaks during fill.
- [ ] Confirm Fuel Transfer Tank Vent is closed.
- [ ] Confirm fill line disconnect completed safely.
- [ ] Return system to Armed state after fill via Mission Control.

## Cryogen Fill Phase
- [ ] **TELL LOX FILL OPERATOR TO DON CRYOGEN PPE.**
- [ ] Confirm non-essential personnel are cleared from LOS.
- [ ] **TELL AVIONICS MANAGER TO ENERGIZE DAQ SENSE POWER.**
- [ ] **TELL ECOMM/ACTCOMM TO CONDUCT COMMS CHECK.**
- [ ] **TELL LOX FILL OPERATOR TO BEGIN LOX DEWAR CONNECTION AND VALVE ALIGNMENT PER CRYOGEN FILL PROCEDURE.**
- [ ] Confirm fill line vent CLOSED.
- [ ] Confirm tank fill valve OPEN.
- [ ] Confirm tank vent valve OPEN.
- [ ] Confirm emergency vent CLOSED.
- [ ] Monitor Cryogen Fill on pad.
- [ ] Monitor gauges for abnormal pressure behavior.
- [ ] Confirm dewar valve closed after fill.
- [ ] Confirm transfer line vented prior to disconnect.
- [ ] **BREAK HERE. WAIT FOR LOX FILL OPERATOR TO CONFIRM FILL.**

## Pad Closeout
- [ ] Confirm all QDs are properly seated.
- [ ] Confirm pneumatic lines connected correctly.
- [ ] **TELL AVIONICS MANAGER TO ENERGIZE DAQ POWER AND MONITOR CURRENT.**
- [ ] **TELL AVIONICS MANAGER TO PERFORM REMOTE SIGNAL CHECK.**
- [ ] **TELL AVIONICS MANAGER TO DE-ENERGIZE GROUND SYSTEM.**
- [ ] **TELL GN2/HIGH PRESS FILL OPERATOR TO SET REGULATORS.**
- [ ] Confirm igniter is connected and isolated properly.
- [ ] **TELL AVIONICS MANAGER TO ENERGIZE SYSTEM.**
- [ ] **TELL MISSION CONTROL MANAGER ALL PAD TEAMS ARE CLEAR.**
- [ ] **BREAK HERE. WAIT FOR HOTFIRE COMPLETION.**

## Abort Authority
- [ ] ABORT if sensor connectivity is lost.
- [ ] ABORT if unintended valve or actuation occurs.
- [ ] ABORT if pressurization does not stabilize.
- [ ] ABORT if passive relief valve actuates unexpectedly.
- [ ] ABORT if any unsafe or off-nominal condition is observed.

## System Safing
- [ ] **BREAK HERE. WAIT FOR MISSION CONTROL MANAGER TO CONFIRM SYSTEM BELOW 25 PSI.**
- [ ] **TELL AVIONICS MANAGER TO DE-ENERGIZE POWER ELECTRONICS.**
- [ ] Confirm manual vents are opened by Fluid System Technician.
- [ ] Inspect fluid system components on static fire pad.
- [ ] Confirm no leaks or hardware damage.
- [ ] **TELL MISSION CONTROL MANAGER PAD IS INSPECTED AND SAFE.**
