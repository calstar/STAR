# Feed System Manager Checklist

*Role: Main manager of the feed system (engine) and GSE. Checks all valves, connections, and reliefs on the engine. Oversees prop operations during hot fire.*

## Pre-test Checkout
- [ ] Execute fluid system diagnostics on the engine and relevant GSE.
- [ ] Verify all non-electrical valves, connections, and reliefs on the engine.
- [ ] Conduct final check for extraneous fittings and verify regulator/check valve orientation.
- [ ] Oversee all propellant operations leading up to the test phase.
- [ ] **TELL PAD MANAGER/MISSION CONTROL ENGINE FEED SYSTEM IS GO.**

## Active Propellant Operations
- [ ] Oversee propellant operations during fill phases.
- [ ] **BREAK HERE. WAIT FOR MISSION CONTROL TO INITIATE HOTFIRE PROTOCOL.**
- [ ] Actively manage feed system operational parameters during hotfire.
- [ ] Monitor line pressures and feed dynamic interactions using telemetry provided by Mission Control/ECOMM.

## Post-Test / Safing
- [ ] Monitor feed system lines for blowdown and safing parameters.
- [ ] Execute manual vents and drain through GSE dumps as required when system drops below 25 PSI.
- [ ] **TELL PAD MANAGER ENGINEERING/FEED SAFING IS COMPLETE.**
