# Avionics Manager Checklist

*Role: Main manager of all electrical systems. Overviews setup, contact with Mission Control Manager.*

## Electrical Setup & Overview
- [ ] Overview the setup of electrical power systems and DAQ systems.
- [ ] Coordinate with ECOMM and ACTCOMM to ensure connections are proper.
- [ ] Establish communication line with Mission Control Manager.
- [ ] **TELL MISSION CONTROL MANAGER ALL ELECTRICAL SYSTEMS ARE SET UP.**

## Pre-Test Procedures
- [ ] **BREAK HERE. WAIT FOR OPERATIONS MANAGER TO GIVE THE GO TO BEGIN.**
- [ ] Energize DAQ SENSE power upon command.
- [ ] Verify ABORT SYSTEM opens tank vents during dry run with ECOMM.
- [ ] Perform Remote Signal Check after Pad team clears.
- [ ] **TELL MISSION CONTROL MANAGER AVIONICS SIGNAL IS NOMINAL.**

## Hotfire Operations
- [ ] Enforce standby protocol across electrical operators (ECOMM, ACTCOMM).
- [ ] Monitor power consumption and DAQ operations during pressurization.
- [ ] **TELL MISSION CONTROL MANAGER AVIONICS IS STEADY FOR IGNITION.**
- [ ] **BREAK HERE. WAIT FOR MISSION CONTROL MANAGER TO TELL YOU TO DE-ENERGIZE WHEN SAFE.**

## System Safing
- [ ] De-energize Power Electronics on command from Mission Control / Pad Manager.
- [ ] Verify system data file saves with ECOMM.
- [ ] **TELL MISSION CONTROL MANAGER AVIONICS ARE DE-ENERGIZED AND DATA SAVED.**
