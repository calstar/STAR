# Operations Manager Checklist

*Role: Main safety manager. Enforces checklists and procedures. Primary POC with FAR range staff.*

## Setup
- [ ] Confirm all personnel are wearing required PPE per role and that spare PPE is available.
- [ ] Confirm Safety Officer and Pad Manager are present, briefed, and ready.
- [ ] Conduct safety briefing with all key personnel:
  - [ ] Review hazard zones, minimum safe distances, and evacuation routes.
  - [ ] Review emergency response plan and abort communication phraseology.
  - [ ] Assign primary and backup POCs for each role.
- [ ] Establish initial POC with FAR range staff and confirm:
  - [ ] Current range status.
  - [ ] Communication channel and call signs.
  - [ ] Range abort and cease-operations protocol.
- [ ] Confirm with Pad Manager and Feed System Manager that rocket-side hoses/pipes/harnessing routing checks are complete.
- [ ] Confirm with GSE Manager, Avionics Manager, ACTCOMM, and ECOMM that GSE-side hoses/pipes/harnessing routing checks are complete.
- [ ] **BREAK HERE. WAIT FOR ALL PERSONNEL TO BE IN DESIGNATED SAFE AREAS AND PAD SAFETY OFFICER CHECKOFFS TO COMPLETE.**
- [ ] Receive confirmation from Safety Officer that pad inspections and checkoffs are complete.
- [ ] **TELL PAD MANAGER YOU ARE GO FOR PAD SETUP.**

## Dry Run & Testing
- [ ] Coordinate timing and test plan with Mission Control Manager and Pad Manager.
- [ ] Confirm all required roles are at their stations and on comms.
- [ ] Require that each role explicitly states checklist section names during callouts (no skipping steps).
- [ ] Ensure checklists are followed in written order without deviation unless a deliberate, documented change is approved.
- [ ] Require verbal GO/NO-GO confirmations from:
  - [ ] Mission Control Manager
  - [ ] Pad Manager
  - [ ] GSE Manager
  - [ ] Avionics Manager
  - [ ] Safety Officer
  before entering any simulated pressurization or actuation during dry run.
- [ ] **TELL MISSION CONTROL MANAGER AND PAD MANAGER WHEN DRY RUN IS STARTING.**
- [ ] At completion of dry run, conduct a short debrief:
  - [ ] Capture any anomalies, miscommunications, or procedural gaps.
  - [ ] Decide if another dry run is required before moving to Press Proof.
  - [ ] **TELL ALL ROLES WHETHER YOU ARE GO OR NO-GO TO PROCEED TO PRESS PROOF.**

## Press Proof
- [ ] Prioritize safety checks before pressurization.
- [ ] Confirm clear zone before pressurization.
- [ ] Confirm safe distance protocol enforced.
- [ ] Confirm all non-essential personnel are outside hazard boundary.
- [ ] Confirm with Pad Manager and Safety Officer that:
  - [ ] Pad is clear.
  - [ ] All tools and loose items are removed from pressurized hardware.
  - [ ] Relief paths and vents are unobstructed.
- [ ] Confirm with GSE Manager that regulator setpoints match the planned Press Proof conditions.
- [ ] Confirm with Mission Control Manager that remote pressurization controls and abort controls are armed.
- [ ] **TELL MISSION CONTROL MANAGER CLEAR ZONE AND SAFETY CRITERIA ARE SATISFIED FOR PRESS PROOF AND AUTHORIZE PRESSURIZATION.**
- [ ] Standby on comms to:
  - [ ] Acknowledge Mission Control Manager reports of tank pressures and stability.
  - [ ] Call ABORT if any safety concern is observed or reported.
- [ ] Once Mission Control Manager reports stable pressures and all roles report nominal, **TELL ALL TEAMS PRESS PROOF IS COMPLETE AND TO TRANSITION TO VENTING / SAFING AS PER MASTER PROCEDURE.**

## Fill
- [ ] Prioritize safety checks before:
  - [ ] Fuel Fill
  - [ ] Cryogen Fill
- [ ] Confirm clear zone before fluid fills.
- [ ] Confirm with:
  - [ ] Pad Manager that pad teams are in proper PPE and only essential personnel are at the pad.
  - [ ] GSE Manager that all fill lines, regulators, and vents are configured per procedure.
  - [ ] Ethanol Fill Operator and LOX Fill Operator that they understand start/stop conditions and abort criteria.
- [ ] **TELL MISSION CONTROL MANAGER AND PAD MANAGER THAT CLEAR ZONE AND PPE ARE VERIFIED FOR FUEL FILL, THEN AUTHORIZE FUEL FILL START.**
- [ ] After Fuel Fill completion and verification:
  - [ ] Receive confirmation from Ethanol Fill Operator and Pad Manager that fuel fill is complete and QDs are disconnected.
  - [ ] **TELL MISSION CONTROL MANAGER FUEL FILL IS COMPLETE AND SYSTEM STATE IS ARMED / AS EXPECTED.**
- [ ] Before Cryogen Fill:
  - [ ] Confirm all non-essential personnel are out of LOS and outside cryogenic hazard boundary.
  - [ ] Confirm LOX Fill Operator and Safety Officer are in full cryogenic PPE.
  - [ ] Confirm LOX vent paths are clear and oriented safely.
- [ ] **TELL MISSION CONTROL MANAGER AND PAD MANAGER THAT CLEAR ZONE AND PPE ARE VERIFIED FOR CRYOGEN FILL, THEN AUTHORIZE LOX FILL START.**
- [ ] After LOX Fill completion:
  - [ ] Receive confirmation from LOX Fill Operator and Pad Manager that LOX fill is complete, lines are vented, and QDs are disconnected or safed.
  - [ ] **TELL MISSION CONTROL MANAGER THAT ALL FILL OPERATIONS ARE COMPLETE AND PAD IS CONFIGURED FOR HOTFIRE.**

## Hotfire
- [ ] Monitor for any deviation from expected procedure flow.
- [ ] **EXECUTE ABORT COMMUNICATION IMMEDIATELY IF SAFETY IS COMPROMISED.**
- [ ] Maintain real-time coordination with FAR range staff during pressurization and burn.
- [ ] Conduct final GO/NO-GO poll prior to ignition, requiring explicit verbal GO/NO-GO from:
  - [ ] Mission Control Manager
  - [ ] Pad Manager
  - [ ] GSE Manager
  - [ ] Avionics Manager
  - [ ] ECOMM
  - [ ] Safety Officer
  - [ ] Feed System Manager
- [ ] **ONLY GIVE IGNITION AUTHORIZATION WHEN ALL POLLED ROLES REPORT GO AND SAFETY CRITERIA ARE SATISFIED.**
- [ ] During hotfire, monitor comms for:
  - [ ] Any anomaly callouts from Mission Control, Pad, ECOMM, Avionics, or GSE.
  - [ ] Any range constraint from FAR staff.
- [ ] If any safety concern or range constraint is raised, **CALL “ABORT, ABORT, ABORT” AND DIRECT MISSION CONTROL MANAGER TO EXECUTE ABORT.**
- [ ] After burn completion, confirm with Mission Control Manager that:
  - [ ] System has been commanded to VENT / ABORT as per master procedure.
  - [ ] All tank pressures are trending down as expected.
- [ ] Confirm system safing parameters met (< 25 psi confirmed by Mission Control).
- [ ] Authorize personnel return to pad only after:
  - [ ] Mission Control Manager confirms all tanks below 25 psi.
  - [ ] Pad Manager confirms vents and manual safing actions as per procedure.
- [ ] Report conclusion of firing and any anomalies to FAR range staff.
- [ ] Document procedural deviations or safety concerns.
