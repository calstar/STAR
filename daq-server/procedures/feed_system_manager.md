# Feed System Manager Checklist

*Role: Main manager of the engine feed system and associated GSE plumbing. Verifies all valves, connections, regulators, and reliefs on the engine. Oversees propellant operations during hotfire.*

## Setup
- [ ] Execute full fluid system diagnostics on engine and relevant GSE.
- [ ] Inspect all non-electrical valves on engine for correct orientation and function.
- [ ] Verify regulator orientation (HP to HP, LP to LP).
- [ ] Verify check valves are installed in correct flow direction.
- [ ] Inspect all Swagelok, NPT, and JIC fittings for proper tightness (no overtightening).
- [ ] Confirm proper thread sealant application on NPT fittings.
- [ ] Verify all relief valves are installed, unobstructed, and venting safely.
- [ ] Confirm no extraneous fittings or temporary hardware remain on engine.
- [ ] Verify engine mounting hardware is secure.
- [ ] Confirm propellant line routing matches P&ID.
- [ ] Verify rocket-side hoses, hard lines, and associated harnessing are routed per P&ID and secured (no kinks or abrasion, clear of sharp edges, hot surfaces, vents, and moving structures).
- [ ] Oversee all propellant operations leading up to test phase.
- [ ] **TELL PAD MANAGER / MISSION CONTROL ENGINE FEED SYSTEM IS GO.**

## Dry Run & Testing
- [ ] Coordinate with Mission Control Manager and GSE Manager to understand planned valve/pressure sequences for dry run.
- [ ] Observe commanded valve and solenoid actions and confirm:
  - [ ] Correct valves move for each commanded state.
  - [ ] No unexpected movement in feed lines or supports.
  - [ ] No leaks or abnormal noises appear when cycling.
- [ ] Confirm with ECOMM that feed-system pressures and any temperatures change as expected during states and return to nominal at the end of dry run.
- [ ] **TELL PAD MANAGER / MISSION CONTROL MANAGER WHETHER FEED SYSTEM IS GO OR NO-GO FOR PRESS PROOF.**

## Press Proof
- [ ] Oversee propellant operations leading into test configuration.
- [ ] Monitor for leaks, frost anomalies, or abnormal line behavior.
- [ ] Confirm fill valve, vent valve, and isolation states match procedure.
- [ ] **TELL PAD MANAGER / MISSION CONTROL MANAGER FEED SYSTEM PRESS PROOF CHECKS ARE COMPLETE AND NOMINAL (OR REPORT ANY ANOMALIES).**

## Fill
- [ ] Oversee propellant operations during Fuel Fill.
- [ ] Oversee propellant operations during LOX Fill.
- [ ] Monitor for leaks, frost anomalies, or abnormal line behavior during fills.
- [ ] Confirm fill valve, vent valve, and isolation states match procedure throughout fill.
- [ ] **BREAK HERE. WAIT FOR MISSION CONTROL TO INITIATE HOTFIRE ATTEMPT / IGNITION SEQUENCE.**
- [ ] After Fuel Fill and LOX Fill, walk the feed system and visually confirm:
  - [ ] Final valve positions match pre-hotfire configuration.
  - [ ] No leaks, abnormal frost patterns, or unexpected line movements.
- [ ] **TELL OPERATIONS MANAGER, PAD MANAGER, AND MISSION CONTROL MANAGER THAT FEED SYSTEM IS CONFIGURED FOR HOTFIRE (OR REPORT ANY ANOMALIES).**

## Hotfire
- [ ] Confirm feed system in correct configuration for pressurization.
- [ ] Actively manage feed system operational parameters during hotfire.
- [ ] Monitor line pressures and feed dynamics via telemetry from Mission Control / ECOMM.
- [ ] Watch for pressure oscillations, cavitation indicators, or unexpected drops.
- [ ] Be prepared to recommend ABORT if feed behavior is off-nominal.
- [ ] Monitor feed system lines during blowdown.
- [ ] Confirm vent valves actuate as commanded.
- [ ] Verify pressure decay trend is nominal.
- [ ] Once system is below 25 PSI, execute manual vents as required.
- [ ] Drain propellants through GSE dump valves as required.
- [ ] Inspect feed system components for leaks or hardware damage.
- [ ] **TELL PAD MANAGER ENGINEERING / FEED SAFING IS COMPLETE.**
